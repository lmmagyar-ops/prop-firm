# .agents/

Internal AI-assisted development tooling configuration.

Contains:
- **`skills/`** — Project orientation guides used by AI coding assistants (architecture context, debugging protocols)
- **`workflows/`** — Standardized runbooks for common development tasks (deploy, bug-fix, financial verification, UI smoke tests)

These files document the operational standards and verification checklists enforced during development. They are kept in source control so the development workflow is reproducible and consistent across sessions.

They are **not** application code and have no effect on the runtime build.
